# DQN — 深度 Q 网络详解

> [English version](../en/dqn-explained.md)

## 什么是 DQN？

**DQN = Deep Q-Network（深度 Q 网络）。** 一种让 AI 通过数百万次试错学会玩游戏的技术——无需人类编写规则。

## 逐词拆解

### Q = 质量（Quality）

游戏中每个动作都有一个"质量分数"——当前有多好。

| 场景：DOOM 中面对敌人 | |
|---|---|
| 动作 | Q 分数 |
| 🔫 射击敌人 | **+8.3** ← 最佳 |
| 🏃 靠近敌人 | +2.1 |
| 😶 站着不动 | -4.7 |
| 🏃 逃跑 | -1.2 |

Q 值越高 = 选择越好。AI 的任务就是学会这些分数。

### Network = 一个小型数学函数

输入游戏像素 → 运行数学计算 → 输出 Q 分数：

```
[120×68 像素图像]
       ↓
  ┌─────────┐
  │  数学   │  （不到 5 毫秒完成）
  └─────────┘
       ↓
  Q(射击)=8.3, Q(移动)=2.1, Q(站立)=-4.7, ...
```

### Deep = 多层数学

"深度"仅指函数有多层计算，而非只有一层。层数越多 = 能学到的模式越复杂。

## DQN 如何学习

### 训练循环

```
1. AI 观察游戏画面
2. AI 选择一个动作（有时随机、有时选当前认为最好的）
3. 游戏给出结果——血量变了吗？杀敌了吗？
4. AI 获得一个"奖励"数值：
     +0.01  每帧存活
     +1.0   击杀敌人
     +0.1   拾取道具
     -5.0   死亡
5. AI 更新 Q 分数："在那个场景，那个动作值 +1.0 分"
6. 重复数百万次
```

### 实例——AI 实际学会了什么

| 第几次 | AI 做了什么 | 结果 | AI 学到的 |
|-------|---------|--------|-----------|
| 1 | 站着不动 | 被打死，奖励=-5.0 | "这里站着不动 = 坏" |
| 2 | 往前走 | 撞墙，无变化 | "撞墙 = 没进展" |
| 3 | 走向敌人，不开火 | 被打死 | "需要射击敌人" |
| … | … | … | … |
| 5万 | 靠近敌人，开火 | 敌人死亡！+1.0 | "射击敌人 = 好！" |
| 10万 | 打死敌人，捡血包 | +0.2（血量恢复） | "血包 = 有用" |
| 50万 | 清空房间，找钥匙，开门 | 多项奖励 | "钥匙 → 门 → 进度" |

经过 **50 万+局游戏**（在 PC 上，不在手机上），AI 形成了内在"直觉"——一个 3 MB 的数字文件，编码着：*"看到这种画面，最好的操作可能是这个。"*

## 输出：一个 3 MB 的 `.tflite` 文件

该文件安装在手机上。无需联网。无需云端。无需 GPU。手机处理器每帧不到 5 毫秒即可运行：

```
游戏运行于 35 fps
      ↓
每 4 帧：截屏 → TFLite 模型推理 → 选最佳动作 → 注入按键
      ↓
AI 每秒做出约 8 次决策
```

## 为什么不用 ChatGPT / 大语言模型？

| | DQN | ChatGPT 类 (LLM) |
|---|---|---|
| 输入 | 原始像素 (120×68) | 屏幕文字描述 |
| 处理 | 1 次前向传播，<5ms | 100+ 步骤串行，5 秒+ |
| 速度 | 实时（可达 30 fps） | 无法实时 (<1 fps) |
| 模型大小 | 2-5 MB | 2,000+ MB |
| 训练 | 从游戏过程中学习 | 从互联网文本学习 |
| 手机上运行 | ✅ 能 | ❌ 不能（太大太慢） |

DOOM 是视觉/空间游戏。不需要用语言"读懂"屏幕——需要识别形状（敌人、墙壁、道具）并快速反应。DQN 天生擅长。用 LLM 就像每帧都用文字描述画面、再读描述、再打字输出指令——太慢了。

## 自我进化（第四阶段）

预训练模型装到手机上后，还能继续进步：

```
你看着 AI 打游戏的过程中：
  ┌──────────────────────────────┐
  │ AI 游玩 → 获得奖励 → 更新    │
  │ 记忆 → 下次同一关卡更快      │
  │ / 更少死亡                   │
  └──────────────────────────────┘
```

经过数小时/数天，AI 在你拥有的 WAD 关卡上慢慢变强。初始训练之后不再需要 PC。

## 术语速查

| 术语 | 大白话 |
|------|---------------|
| **DQN** | "试错几百万次来学习" |
| **Q 值** | "这个动作现在有多好？"（越高越好） |
| **奖励** | 一个数字：+1 杀敌，-5 死亡 |
| **神经网络** | 一个小型数学函数（3 MB，5 毫秒） |
| **训练** | PC 上打 50 万+局来构建这个函数 |
| **推理** | 手机上实时运行这个函数 |
| **自我进化** | 手机在你观看时继续学习 |
| **TFLite** | TensorFlow Lite — Google 的手机 AI 运行工具 |
